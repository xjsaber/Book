irb实际上是Ruby的“shell”，类似于操作系统的shell（具备任务控制）

	irb [irb选项] [ruby脚本] [程序参数]

## 13.3 编辑器支持 Editor Support ##
Ruby解释器被设计成一次读一个程序：意味着可以通过管道把整个程序作为标准输入传递给解释器，而它就可以很好的工作。
