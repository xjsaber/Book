## 3.4 变量 ##
在Java中，每一个变量属于一种类型（type）。在声明变量时，变量所属的类型位于变量名之前。

### 3.4.2常量 ###
利用关键字final指示常量。
关键字final表示这个变量只能被赋值一次。一旦被赋值之后，就不能够再更改了。习惯上，常量名使用全大写。
