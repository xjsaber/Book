### 建议1：不要在常量和变量中出现易混淆的字母 ###

包名全小写，类名首字母全大写，常量全部大写并用下划线分隔，变量采用驼峰命名法（Camel Case）命名等。

字母“l”作为长整型标志时务必大写。

### 建议2：莫让常量蜕变成变量 ###
