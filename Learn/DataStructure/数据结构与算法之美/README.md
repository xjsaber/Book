### 时间复杂度分析 ###

1. 只关注循环次数最多的一段代码
2. 加法法则：总复杂度等于量级最大的那段代码的复杂度
3. 乘法法则：嵌套代码的复杂度等于嵌套内外代码复杂度的乘积。

### 几种常见时间复杂度实例分析 ###

* 常量阶O(1)
* 对数阶O(log^n)
* 线性阶O(n)
* 线性对数阶O(nlog^n)
* 指数阶O(2^n)
* 阶乘阶O(n!)
* 平方阶O(n^2)、立方阶O(n^3)....K次方阶O(n^k)

### 内容小结 ###

O(1)、O(log^n)、O(n)、O(nlog^n)、O(n^2)
