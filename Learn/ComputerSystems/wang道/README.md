### 1.1 操作系统的基本概念 ###

#### 1.1.1 操作系统的概念 ####

计算机系统自下而上可大致分为4部分：硬件、操作系统、应用程序和用户

操作系统（OS）是指控制和管理整个计算机系统的硬件与软件资源，合理地组织、调度计算机的工作与资源分配，进而为用户和其他软件提供方便接口与环境的程序集合。操作系统是计算机系统中最基本的系统软件。

#### 1.1.2 操作系统的特征 ####

**1. 并发（Concurrence）**

并发是指两个或多个事件在同一时间间隔内发生。

2. 共享（Sharing）

3. 虚拟（Virtual）

4. 异步（Asynchronism）
