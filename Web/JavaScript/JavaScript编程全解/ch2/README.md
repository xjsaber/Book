## 2.4 函数基础 ##

### 2.4.1 函数的定义 ###
通常情况下，JavaScript代码是自上而下执行的，不过函数体内部的代码则不是这样。如果只是对函数进行了声明，其中的代码并不会执行。只有在调用函数时才会执行函数体内部的代码。
	console.log('1');
	function f() {
		console.log('2');
	}
	console.log('3')
	f();
