## 4.21 continue语句 ##
在循环中使用continue语句的话，就会跳过在此之后本次循环内尚未执行的语句，而返回至循环条件表达式进行求值。

## 4.22 通过标签跳转 ##
在嵌套循环的内层使用break语句的话，仅仅会跳出内层的循环。

## 4.23 return语句 ##
return语句会中断函数的处理，并将指定的表达式的值作为函数的返回值返回。

## 4.24 异常 ##
可以通过throw语句来抛出异常对象（异常值）。
