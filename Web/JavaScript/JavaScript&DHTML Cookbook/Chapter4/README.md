## 4.7 漂亮地处理脚本错误 ##

#### 问题 ####

#### 解答 ####
`Math.random()`返回0到1之间的一个伪随机数字。

	var result = Math.floor(Math.random() * (n + 1));
这里的n是这个范围的上限。

	var result = Math.floor(Math.random() * (n + 1)) + m;
这里的m是范围的下限，而n是范围的上限。
