### 2.1 建立WebRTC会话 ###

建立WebRTC会话时，需要以下四个主要步骤：

1. 获取本地媒体
2. 在浏览器和对等端（其他浏览器或终端）之间建立连接。
3. 将媒体和数据通道关联至该连接。
4. 交换会话描述。
