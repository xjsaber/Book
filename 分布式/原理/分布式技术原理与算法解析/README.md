* 相同的应用部署到不同的服务器上，当大量用户请求过来时，如何能比较均衡地转发到不同的应用服务器上？设计一个负载均衡器
* 当请求量较大时，对数据库的频繁读写请求，使得数据库IO访问成为瓶颈。解决这个问题的方式是读写分离，读数据库只接受读请求，写数据库只接受写请求，当然读写数据库之间要进行数据同步，以保证数据一致性。
* 当有些数据成为热点的数据时。使用缓存
