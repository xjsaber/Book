## 1.2 关于指针 ##

### 1.2.1 恶名昭著的指针究竟事什么 ###

指针是一种保存变量地址的变量，在C中频繁地使用。

指针类型（pointer type）可由函数类型、对象类型或不完全地类型派生，派生指针类的类型称为引用类型。指针类型描述成一个对象。
