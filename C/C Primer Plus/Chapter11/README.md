### 11.2.2 不幸的gets()函数 ###
如果输入的字符串过长，会导致缓冲区溢出（buffer overflow），即多余的字符超出了制定的目标空间。如果这些多余的字符只是占用了尚未使用的内存，就不会立即出现问题；如果它们擦写掉程序中的其他数据，会导致程序异常中止；或者还有其他情况。
